# project2 — Books API

Breve: Exemplo de API REST para gerenciar livros, usada como exercício no curso de FastAPI.

Execução:

```powershell
cd project2-books_complete_restful_api
uvicorn books:app --reload --port 8001
```

Pontos importantes:
- Arquivo principal: `books.py`.
- Documente endpoints e modelos conforme adicionar novas rotas.

Testes:
- Adicione testes em uma pasta `test/` e execute com `pytest`.
